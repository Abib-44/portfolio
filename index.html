Voici quelques-uns de mes projets récents

Projet Airflow-Docker

Le projet Airflow-Docker est un système d'orchestration de workflows construit avec Apache Airflow, utilisant PostgreSQL comme base de données et pgAdmin pour la gestion. Le projet est entièrement conteneurisé avec Docker, offrant un environnement isolé et reproductible pour les pipelines de données et la planification des tâches.

Dépôt GitHub

Pipeline d'Automatisation CSV

La Pipeline d'Automatisation CSV est un projet qui nettoie et traite automatiquement des fichiers CSV en utilisant Python et Pandas. Le système fonctionne dans des conteneurs Docker et stocke les données purifiées directement dans une base de données SQLite, offrant un flux de travail efficace et reproductible pour la gestion et l'automatisation des données.

Dépôt GitHub

CI/CD Python avec Docker

Ce projet met en place un pipeline CI/CD complet pour des applications Python, utilisant Docker pour l'isolation des environnements. Il automatise les tests, la construction et le déploiement, intégrant MinIO pour la gestion du stockage et assurant un flux de travail reproductible et fiable pour le développement continu.
